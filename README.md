# konawiki3

Wiki clone

Konawiki --- http://kujirahand.com/konawiki/

## Konawiki3 is simple.

It is very simple PHP wiki engine.

``
* header

text text text text

| table | test
| aaa | bbb
``

## Mark down also can use

It also can write with mark down.

When you want to write mark down, you should execute command:

```
$ cd kona3engine
$ composer install
```
